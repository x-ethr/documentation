# `/edit-team` command

Change your team's name and/or color.

**Additional Options**
- `name` *(String)* *Optional* - The new name of the team. Must be between 1-35 chars
- `color` *(String)* *Optional* - A new color in hexcode format. ex: `#FFFFFF`. To get a random color, write 'random'
