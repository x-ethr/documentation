# `/demerit` commands

All demerit related commands.

<Callout type={"info"}>It is recommended to only have `Organizers` or moderators use this command.</Callout>

<br/>

## `/demerit give`

Give a demerit to a user (these are tracked)

**Required Options**
- `user` *(User)* - The target user
- `reason` *(String)* - The reason for the demerit

<br/>

## `/demerit list`

List all demerits that a user has

**Required Options**
- `user` *(User)* - The target user
