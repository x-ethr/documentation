# `/timeout` command

Timeout a user. This command leverages Discord's built-in timeout system, however, you can specify granular timeout
durations.


**Required Options**
- `user` *(User)* - The target user
- `reason` *(String)* - The reason for the timeout

**Additional Options**
- `minutes` *(Number)* *Optional* - Number of minutes
- `hours` *(Number)* *Optional* - Number of hours
- `days` *(Number)* *Optional* - Number of days
