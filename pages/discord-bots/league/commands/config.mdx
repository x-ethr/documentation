# `/config` command

Bot configuration command.

## Defaults

| Config Value    | Default             |
|-----------------|---------------------|
| regions         | `NA,SA,EU,OCE,ASIA` |
| max-roster-size | `10`                |


**Additional Options**
- `regions` *(String)* *Optional* - Comma separated list of region values to enable. Ex: `NA,SA,EU,OCE,ASIA`
- `max-roster-size` *(String)* *Optional* - Maximum size each team's roster can reach. Ex: `10`
